3 步开始使用

简单快速，即刻体验

1

获取扩展

从 Chrome Web Store 安装，或从 GitHub 下载源代码

2

加载到 Chrome

打开 chrome://extensions/，开启开发者模式，加载已解压的扩展程序

3

开始整理

打开新标签页即可看到书签白板，开始拖拽添加你的第一个书签！

从 Chrome 商店安装 从 GitHub 安装 查看使用文档
