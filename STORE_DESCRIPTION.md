## 🔐 权限说明

| 权限 | 用途 |
|------|------|
| **storage** | 本地存储书签数据，确保数据持久化保存 |
| **contextMenus** | 添加右键菜单，方便快速添加书签 |
| **tabs** | 获取当前标签页信息，用于一键添加当前页面 |
| **scripting** | 注入脚本获取网页标题，提升用户体验 |
| **sidePanel** | 提供侧边栏功能，快速访问书签 |

**注意**：本扩展无需 host_permissions，所有跨域请求均在扩展页面和 service worker 中执行。

## 📋 隐私政策摘要

**数据收集**：本扩展**不会收集**任何用户数据。

**数据存储**：所有数据完全保存在本地浏览器中（Chrome Storage Local），不上传服务器。

**数据使用**：仅用于提供核心功能，不与第三方共享。

**完整隐私政策**：https://github.com/YinHeng89/bookmark-board
