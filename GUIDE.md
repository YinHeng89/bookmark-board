## 🚀 快速开始

### 安装扩展

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择书签白板项目文件夹
6. ✅ 安装完成！

### 首次使用

- **打开新标签页** → 自动显示书签白板主界面
- **点击工具栏图标** → 打开侧边栏
- **右键任意网页** → 添加到书签白板

## ✨ 核心功能

### 添加书签

书签白板提供 **5 种添加方式**，满足各种场景需求：

#### 1️⃣ 拖拽添加
- 从地址栏拖拽 URL 到书签区域
- 从网页拖拽链接到书签区域或侧边栏
- 从浏览器书签栏拖拽书签到书签区域
- ✅ **自动提取标题和图标**

#### 2️⃣ 右键菜单 - 添加页面
- 在网页空白处右键
- 选择「添加到书签白板」
- ✅ **自动保存当前页面**

#### 3️⃣ 右键菜单 - 添加链接
- 在网页链接上右键
- 选择「添加链接到书签白板」
- ✅ **自动保存该链接**

#### 4️ 侧边栏一键添加
- 打开侧边栏（点击工具栏图标）
- 点击 ➕ 按钮
- ✅ **自动添加当前浏览页面**

#### 5️ 手动添加
- 点击「手动添加」按钮
- 输入 URL（必须包含 `https://` 或 `http://`）
- ✅ **支持添加任意链接**

---

### 管理书签

####  编辑书签
- 点击卡片上的编辑按钮
- 修改书签名称
- 支持 Enter 键确认，ESC 键取消

#### 🗑️ 删除书签
- 点击卡片上的删除按钮
- 弹窗确认，防止误删
- 支持 ESC 键取消删除

#### ️ 打开书签
- **单击卡片** → 在新标签页打开链接
- **自动记录访问次数和最后访问时间**

#### 📌 置顶书签
- 右键点击书签卡片
- 选择「置顶此书签」
- 置顶书签会在「置顶」分区优先显示

---

### 搜索与筛选

#### 🔍 实时搜索
- 在搜索框输入关键词
- 即时过滤书签标题和 URL
- 支持中英文搜索

#### 🏷️ 分组搜索
- 输入 `!` 或 `！` + 分组名称
- 例如：`!工作` 或 `！工作`
- 快速筛选特定分组的书签

---

### 分区浏览

书签白板提供 **3 个视图分区**，点击顶部 Tab 切换：

#### 📚 所有书签
- 显示全部书签
- 支持分组筛选
- 显示书签总数角标

####  置顶
- 仅显示置顶书签
- 快速访问重要书签
- 空状态提示如何置顶

####  最近添加
- 显示最近 7 天内添加的书签
- 基于 `createdAt` 时间字段
- 自动排除置顶书签

---

### 排序功能

书签白板提供 **5 种排序方式**：

#### ⏱️ 按添加时间（新→旧）
- **默认排序**
- 最新添加的书签显示在最前面
- 适合经常添加新书签的用户

#### ⏱️ 按添加时间（旧→新）
- 最早添加的书签显示在最前面
- 适合查找历史书签

#### 📝 按名称（A→Z）
- 按书签标题字母顺序排序
- 支持中文拼音排序
- 适合按字母查找

#### 📝 按名称（Z→A）
- 按书签标题倒序排序

#### 📊 按使用频率（高→低）
- 点击次数多的书签排在前面
- 基于使用统计自动排序
- 适合快速访问常用书签

**使用方式**：
- 点击顶部排序下拉框
- 选择排序方式
- 即时生效，置顶书签始终在最前

## 📂 分组管理

### 手动分组

#### 创建分组
1. 点击分组栏右侧的  按钮
2. 输入分组名称
3. 点击「确认」或按 Enter 键

#### 编辑分组
1. 右键点击分组标签
2. 选择「编辑分组」
3. 修改名称后确认

#### 删除分组
1. 右键点击分组标签
2. 选择「删除分组」
3. 确认后删除（**不会删除分组内的书签**）

#### 分组书签
- 右键点击书签卡片
- 选择「添加到分组」
- 选择目标分组
- ✅ **支持多分组**（一个书签可属于多个分组）

---

### 自动分组

#### 什么是自动分组？
系统会根据书签的**域名**自动创建分组：
- `github.com` → 自动创建「GitHub」分组
- `google.com` → 自动创建「Google」分组
- `bilibili.com` → 自动创建「哔哩哔哩」分组

#### 自定义自动分组名称
1. 打开**设置页面**
2. 进入「书签管理」→「自动分组」
3. 点击域名后的编辑按钮
4. 输入自定义名称（如：`GitHub` → `代码仓库`）

#### 合并自动分组
- 多个子域名可合并到一个分组
- 例如：`mail.google.com` 和 `drive.google.com` 都可归入「Google」

---

### 分组筛选

#### 快速筛选
- 点击分组标签
- 仅显示该分组的书签
- 再次点击取消筛选

#### 全部分组
- 点击「全部分组」按钮
- 显示所有书签
- 取消分组筛选

#### 分组数量角标
- 每个分组标签右侧显示书签数量
- 实时更新，方便了解分组规模

## 🤖 AI 智能分类

书签白板内置 AI 智能分类功能，基于 OpenAI API 自动为书签分组。

### 配置 AI

#### 1️⃣ 获取 API Key
1. 注册 [OpenAI 账号](https://platform.openai.com/)
2. 进入 [API Keys 页面](https://platform.openai.com/api-keys)
3. 创建新的 API Key
4. 复制保存（只显示一次）

#### 2️⃣ 配置 API
1. 打开**设置页面**
2. 进入「AI 助手」Tab
3. 输入 API Key
4. 选择模型（推荐：`gpt-3.5-turbo`）
5. 保存配置

### 使用 AI 分类

####  自动分类
1. 点击顶部「AI 分类」按钮
2. 选择要分类的书签（或全选）
3. AI 自动分析书签内容
4. 智能推荐分组
5. 确认后自动应用

#### AI 分类特点
- ✅ **智能分析**：基于书签标题和 URL 理解内容
- ✅ **多分组支持**：一个书签可归入多个分组
- ✅ **自动创建**：不存在的分组会自动创建
- ✅ **中文优化**：完美支持中文书签分类

#### 注意事项
- ⚠️ AI 分类需要联网
- ⚠️ 会消耗 OpenAI API 额度
- ⚠️ 首次使用需要配置 API Key
- 💡 建议批量使用，节省 API 调用次数

#### 排序选项

- **标题**（A-Z/Z-A）

#### 最近添加
- 显示最近 7 天内添加的书签

- 自动排除置顶书签

### 设置中的分组管理

### 数据管理

#### 📤 导出数据
- 点击「导出数据」按钮
- 导出完整的 JSON 文件
- **包含数据**：
  - ✅ 所有书签（URL、标题、图标、分组、访问次数）
  - ✅ 所有分组（手动分组 + 自动分组配置）

  - ✅ 排序设置
  - ✅ 自定义自动分组名称

- 点击「导入数据」按钮
- 选择之前导出的 JSON 文件
- 解密并验证数据
- 确认后**覆盖当前所有数据**
- ⚠️ **警告：导入会覆盖现有数据，请谨慎操作！**

#### 数据统计
- 书签总数

## 💾 数据导入导出

### 导出步骤

1. 进入**设置页面**
2. 切换到「数据管理」Tab
3. 点击「导出数据」按钮
4. 自动下载加密的 JSON 文件
5. 保存到安全位置

### 导入步骤

1. 进入**设置页面**
2. 切换到「数据管理」Tab
3. 点击「导入数据」按钮
4. 选择之前导出的 JSON 文件
5. 查看数据统计（书签数、分组数）
6. 确认导入（⚠️ 会覆盖现有数据）
7. 等待导入完成
8. 自动刷新界面

### 加密说明

**四层加密流程**：
```
原始数据 (JSON)
  ↓ UTF-8 编码
字节数组
  ↓ Base64 编码
Base64 字符串
  ↓ XOR 密钥混淆
加密字符串
  ↓ Base64 编码
最终加密文件 (可打印字符)
```

**密钥**：`bookmark-board-2026`  
**安全性**：防止明文泄露，保护用户隐私

## ❓ 常见问题

### Q: 书签数据存在哪里？
A: 所有数据存储在 `chrome.storage.local`，完全本地保存，不上传服务器。

### Q: 清除浏览器数据会丢失书签吗？
A: 会的！清除浏览器数据会删除 `chrome.storage.local` 中的书签。**建议定期导出备份**。

### Q: 如何备份书签？
A: 使用设置页面的「导出数据」功能，保存加密的 JSON 文件。

### Q: 可以在多台电脑同步吗？
A: 当前版本不支持云端同步。可以手动导出导入数据文件实现迁移。

### Q: 右键菜单没有显示？
A: 尝试完全重新安装扩展（移除后重新加载）。

### Q: 侧边栏不自动刷新？
A: 确保使用的是最新版本（v3.2.5+）。如果仍有问题，关闭并重新打开侧边栏。

### Q: 如何修改书签图标？
A: 系统自动获取网站的 favicon.ico，暂不支持手动修改。

### Q: 导入数据后主题变了？
A: 导入文件包含主题设置，会自动应用导出时的主题。可以在设置中重新切换。

### Q: 删除分组会删除里面的书签吗？
A: 不会！删除分组只是移除分组标签，书签本身不会删除。

### Q: 一个书签可以属于多个分组吗？
A: 可以！书签支持多分组归属。
